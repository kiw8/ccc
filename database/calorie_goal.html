Calorie Goal API Documentation

1. Get Calorie Goal

This function retrieves the daily calorie goal for a specific user based on their ID.

        Function Name: getCalorieGoal($userId)
        Method: SELECT
        Parameters:
            - userId (integer): The ID of the user
        Returns:
            - JSON response with daily_calorie_goal or an error message

Example Response:

        {
            "daily_calorie_goal": 2000
        }

2. Update Calorie Goal

This function updates the daily calorie goal for a specific user.

        Function Name: updateCalorieGoal($userId, $daily_calorie_goal)
        Method: UPDATE
        Parameters:
            - userId (integer): The ID of the user
            - daily_calorie_goal (integer): The new daily calorie goal
        Returns:
            - JSON response with success or error message

Example Response:

        {
            "message": "Calorie goal updated successfully"
        }

Database Queries

Get Calorie Goal

        SELECT daily_calorie_goal
        FROM user
        WHERE id = ?;

Update Calorie Goal

        UPDATE user
        SET daily_calorie_goal = ?
        WHERE id = ?;
